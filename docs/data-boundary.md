# 公开与私有数据边界

## 可以进入仓库

- 模型代码、数据 schema、验证规则、合成 fixture；
- 可公开再分发且许可证明确的数据下载器；
- 不含个人权重和金额的参数结构；
- 单元测试、CI、拒绝原因和可复现说明。

## 永远留在本地或受控存储

- 真实持仓、权重、份额、成本、盈亏、税务和交易记录；
- 未归属股份、薪酬、账户标识和家庭资产负债；
- 券商导出、账户截图、私人研报和付费数据原文件；
- `.env`、访问令牌、API key、Cookie 和签名 URL；
- 每日运行产物中可能反推出真实组合的快照或日志。

生产运行通过命令行显式注入本地输入。代码中不得放真实默认值，测试只能使用标注为 `SYNTHETIC` 的确定性数据。

ETF 本地篮子构建器只接受运行时注入的成分、权重和生效日。公开仓库只能保存通用 schema、校验器和合成测试；本地映射、生成的代理收益、持仓快照及其 manifest 都不得提交。公开产物可以记录门禁状态和不可逆哈希，但不得记录可反推出个人成分或权重的字段。
